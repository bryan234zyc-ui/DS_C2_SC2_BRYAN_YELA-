# DS_C2_SC2_BRYAN_YELA-
# Análisis del progreso mundial de vacunación

Este proyecto analiza datos de vacunación mundial utilizando Python y Pandas.

## Contenido

- Lectura de datos desde archivo CSV.
- Revisión de estructura y tipos de datos.
- Conversión de fechas a formato datetime.
- Cálculo de vacunas aplicadas por compañía.
- Cálculo de vacunas aplicadas en todo el mundo.
- Promedio de vacunas aplicadas por país.
- Filtrado de registros por fecha.
- Creación de dataframes derivados.
- Exportación de resultados a un archivo Excel.

## Archivos

- `DS_C2_SC2_BRYAN_YELA.ipynb`: libreta principal del análisis.
- `country_vaccinations.csv`: base de datos utilizada.
- `resultadosReto.xlsx`: archivo con los dataframes generados.
